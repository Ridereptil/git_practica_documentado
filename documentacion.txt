pasos 1 para iniciar el repositorio
git init

paso 2 para verificar de que todo este correcto
git status

paso 3 guardar los cambios 
git add .

paso 4 saber de que git remoto sera en el que se subiran los cambios
git remote add origin https://github.com/Ridereptil/git_practica_documentado.git

paso 5 confirmar los cambios que se hicieron como un punto de guardado
git commit -m "primer commit"

paso 6 subir los cambios al repositorio remoto en la rama que se le indico
git push origin master

paso 7 creas la rama en la cual se haran cambios
git branch nueva

paso 8 cambias de rama a la nueva rama creada
git checkout rama1

paso 9 haces cambios en la segunda rama
git add . 
git commit -m "los cambios"
paso 10 unes la rama creada con la original
git merge nueva
